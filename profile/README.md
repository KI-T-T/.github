# KI.T.T. README

## Outcome Vorbereitungs Workshop

Wir werden das Projekt in zwei Streams aufteilen damit jeder das machen kann was er gerne lernen möchte:

### Advanced Prompting in intelliJ mit Copilot chat bzw. agent
in diesem Stream wollen wir die vorhande Unterstützung von der IDE benutzen mit dem Ziel das wir schnell fortschritte machen. 
Der Nachteil ist, das wir durch die eingebauten SystemPrompts etwas eingeschränkt sind. Wir haben verschiedene LLM's zur Auswahl.
